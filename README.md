# webproj - 기본 게시판
**Node.js, HTML, CSS, JavaScript**

---

## 소개
상품을 구매할 수 있는 쇼핑몰입니다.

---

## 주요 기능
- 회원가입 / 로그인
- 상품 등록 / 판매
- 장바구니 / 상품 구입
- 게시판 및 페이징
